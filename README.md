# Customer-Suppert-Bot
AI Customer Support Bot built using Flowise and OpenRouter with FAQ handling and AI fallback.


AI Customer Support Bot

Overview

This project is a simple and efficient AI-powered customer support assistant built using **Flowise** and **OpenRouter**.

It can:

* Answer predefined FAQs
* Handle unknown queries using AI
* Provide clean and user-friendly responses

---

Features

 ✅ FAQ-based response system
 ✅ AI fallback for unknown queries
 ✅ Clean conversational flow
 ✅ Easy to deploy and use

---

How It Works

The bot uses a simple pipeline:

User Input → Prompt Template → LLM → Response

* If the query matches FAQs → predefined answer
* Else → AI generates a helpful response

---

Tech Stack

* Flowise (Visual AI pipeline builder)
* OpenRouter API (LLM integration)
* Large Language Models (GPT / Mixtral)

---

Flow Design

Prompt Template → LLM Chain → Chat Model

---

Example Queries

* "What is your refund policy?"
* "How long does delivery take?"
* "Hello, how are you?"

---

Setup Instructions

1. Clone the repository
2. Open Flowise
3. Import or recreate the flow
4. Add your OpenRouter API key
5. Run the chatbot
